# Education Registration (dv_education_registration)

Online registration for education courses with email confirmation and inquiry forms.

## Features

- Event registration with confirmation email workflow
- Inquiry forms (inhouse training / info material requests)
- Admin email notifications
- DSGVO-compliant garbage collection for expired records
- Depends on `dv_education_courses`

## Requirements

- PHP 8.2+
- TYPO3 v12.4 LTS or v13.4 LTS
- `davitec/dv-education-courses` ^1.0

## Installation

```bash
composer require davitec/dv-education-registration
```
